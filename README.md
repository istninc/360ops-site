# 360ops Site

Marketing landing page for [360ops.ai](https://360ops.ai) — AI-Edge Operations Platform.

## Stack

Static HTML/CSS/JS served via `serve`.

## Local Dev

```bash
npx serve . -p 3000
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy (Railway)

1. Connect this repo to a new Railway service
2. Railway auto-runs `npm start`
3. Set custom domain `360ops.ai` in Railway service settings
